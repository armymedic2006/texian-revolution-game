# Artillery, Structures, and Defenses

## Artillery Model

Artillery units include:

- Cannon
- Crew
- Crew personal weapons

Crew can:

- Operate cannon
- Fire personal weapons
- Move/reposition eligible artillery
- Count for victory point control

Cannon crew personal weapons use artillery firing priority.

## Artillery Ammunition

Version 1 artillery ammunition types:

- Solid shot
- Grapeshot
- Canister

No artillery ammunition limits in version 1.

The system should support ammunition limits later.

## Artillery Terrain Damage

Only certain ammunition types damage structures.

Likely rule:

- Solid shot damages walls, gates, barricades, doors, buildings, artillery positions, ladders, siege equipment, and defensive earthworks.
- Grapeshot and canister are primarily anti-personnel.

## Artillery Placement

- Both players place artillery during deployment.
- Deployment is hidden from the opposing side.
- 18-pounder and 16-pounder become fixed after initial placement.
- All other guns may be repositioned.
- Lighter artillery repositions faster.
- Artillery automatically rotates/aims when firing.

## Damageable Structures

Included:

- Walls
- Gates
- Barricades
- Doors
- Buildings
- Artillery positions
- Ladders / siege equipment
- Defensive earthworks
- Other relevant battlefield structures

Excluded:

- Roofs

## Structure Damage States

1. Intact
2. Damaged
3. Heavily Damaged
4. Destroyed

Destroyed structures create tactical map changes.

Examples:

- Destroyed wall creates breach
- Destroyed gate opens path
- Destroyed barricade removes cover
- Destroyed door opens entry
- Destroyed artillery position exposes/disables cannon
- Destroyed ladders/siege equipment block assault options

## Defense Construction

Any soldier squad can:

- Repair existing defenses
- Improve existing defenses
- Build simple temporary defenses

Leaders do not directly build but can improve nearby squad construction speed/effectiveness.

Defense work depends on:

- Squad size
- Morale
- Combat effectiveness
- Whether under fire
- Defense type
- Fatigue

No tools/material tracking in version 1.

Design for future resource tracking.

## Temporary Defenses

Both sides can build:

- Light barricade
- Heavy barricade
- Spike/obstacle
- Firing position

Both sides can build assault works:

- Ladders
- Breaching tools
- Siege positions for artillery
- Forward firing positions

Ladders and breaching tools are buildable during battle in version 1.

The data model should support pre-existing equipment later.
