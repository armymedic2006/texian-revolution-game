# Texian Revolution Game

A browser-based alternate-history tactical strategy game focused on the Battle of the Alamo.

Version 1 is scoped to a single playable Battle of the Alamo scenario with:

- Texian defenders vs Mexican army
- Single-player vs AI
- Local/hot-seat multiplayer
- Simultaneous hidden-order planning
- Fog of war
- Morale and fatigue
- Free-radius movement
- Artillery, melee, structures, cover, facing, and victory points

The project is currently in documentation and requirements setup.

See `/docs` for the full Codex-ready requirements package.
